# STM32ADCGPIO_SerialMonitor
STM32 ile ADC ve GPIO okumalarını seri port üzerinden izleyin, istediğiniz dil ile analog dünyaya açılın. 

STM32F103C8T6 Bluepill uyumludur. 

Komut Listesi 
Okuma için '1' karakterini gönderin. 

| Komut | Açıklama |
| '1' | Okuma |
|  'A' | PA10 HIGH |
| 'a' | PA10 LOW |
